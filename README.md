mediawiki-MimeTex
=================

usage MimeTeX for insert math formulas in MediaWiki

Usage
=====
```
 <math>E=mc^2</math>
```
Installations
=============

* Clone repo to extensions folders
* Change imgurl in mimetex.php file
* add require_once("extensions/MimeTex/mimetex.php"); into LocalSettings.php file
